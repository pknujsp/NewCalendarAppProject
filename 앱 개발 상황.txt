-주변 장소
카테고리 검색 목록내 장소표시 정보 확대

건물 정보 : 지도의 좌표값을 넘기면 해당 좌표의 건물 정보가 응답
값 : 건물명, 건물유일 식별자, 건물 최하/최상층, 건물 새주소

건물 층정보 : 건물유일 식별자를 넘기면 건물 층정보가 응답
값 : 건물명, 전개도 유형, 행정도/새 주소, 건물 최하/최상층

건물 층별 시설물 공간 속성 : 건물 유일 식별자와 검색을 원하는 층 번호를 넘기면 해당 층의 시설물 정보가 응답
값 : 시설물 종류

건물 층별 사업체 정보 : 건물 유일 식별자와 건물 층 번호를 넘기면 해당 층에 있는 사업체 정보가 응답
값 : 사업체 정보(좌표, 회사명), 시설정보(시설 좌표, 시설물 종류), 테마 코드 정보(테마 코드, 테마 코드명 = 기업(제조/화학))


- 이벤트/인스턴스 datetime 표시 방식
<allday인 경우>
시간대 값이 필요없으므로 UTC시간 그대로 출력됨
20200101 - 20200110으로 저장하면 UTC시각으로 20200101 0000 - 20200110 0000으로 나온다
<allday가 아닌 경우>
시간대 값이 필요, 지정된 시간대로 출력됨



- 빌딩 층 계산
a : 지상 층수
i : 리스트내 position

if i>=a : floor = i -(a+1) - (i-a)*2
else : floor = a-i
